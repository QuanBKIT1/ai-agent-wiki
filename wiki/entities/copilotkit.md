# CopilotKit

Tổ chức ra mắt **[[ag-ui|AG-UI (Agent-User Interaction Protocol)]]** vào 5/2025 — chuẩn dựa trên event cho cách agent giao tiếp với UI.

AG-UI đã được tích hợp với [[langgraph|LangGraph]], [[crewai|CrewAI]], Mastra, và Microsoft Agent Framework.
